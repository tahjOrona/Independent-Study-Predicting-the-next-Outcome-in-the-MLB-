# Independent-Study-Predicting-the-next-Outcome-in-the-MLB-
This is my Idependent Study that was completed at The College of Wooster. It has the ability to guess the next outcome in baseball. 

You will have to download several packages including Pyton 3.10 or an older version for tensorflow. 
If you don't have this first download homebrew path which is:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/brew.sh/install/HEAD/install.sh)"

then download python 3.10:
brew install python@3.10

Then finally you will have to install pandas, numpy, scikit-learn, tensorflow, and matplotlib
To do this you will use:
pip install pandas numpy scikit-learn tensorflow matplotlib
or pip3 install pandas numpy scikit-learn tensorflow matplotlib
